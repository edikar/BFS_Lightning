# BFS_Lightning
Lightnin sinulator based on BFS search algorithm
Inspired by the following video:
https://www.youtube.com/watch?v=akZ8JJ4gGLs

## 1. Prerequisites
libGL  
libGLEW   
glfw

## 2. Compilation:
 `gcc -std=c99  -o lightning lightning.c -lGLEW -lGL -lglfw;`

## 3. Execution: 
 ./lightning`
